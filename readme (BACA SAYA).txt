*PASTIKAN ANDA TERKONEKSI DENGAN JARINGAN INTERNET PADA SAAT MENGINSTALL*

CARA MENGINSTALL
1. Buka install.bat

CARA MEMBUKA APLIKASI
1. Buka app.py

CARA MENJALANKAN APLIKASI PENDETEKSI JERAWAT (PORESIGHT)
1. Buka app.py
2. Tekan tombol run
3. Tunggu sebentar
4. Ketika aplikasi berhasil dimulai, lihat di terminal untuk IP dan portnya
   Contoh: - http://127.0.0.1:5000
 	   - http://192.168.18.207:5000
5. Copy salah satunya dan paste di browser kesayangan
6. Jika sudah maka akan muncul tampilan aplikasinya
7. Pencet "Choose File" untuk mengupload foto sample untuk dicoba (Bisa pakai foto sendiri, tetapi kami sudah
   menyiapkan beberapa sample di dalam folder PoreSight\coba
8. Tekan Upload untuk mengunggah ke sistem Machine Learning
9. Tunggu proses scanning
10. Gambar yang sudah ditandai akan muncul
11. Aplikasi sudah berhasil digunakan, silakan mencoba sample lain!

CARA MELIHAT HISTORY FOTO UPLOADS
1. Buka folder PoreSight\uploads
2. Foto dapat dilihat di folder ini

CARA MELIHAT HISTORY FOTO HASIL SCANNING
1. Buka folder PoreSight\results
2. Foto dapat dilihat di folder ini
